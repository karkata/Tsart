Tsart 기본 사용법

Tsart는 챠트 라이브러리를 사용하려면 tsart.js 파일과 tsart-{type}.js 파일을 script 태그에 선언을 해야 한다. tsart.js 파일은 Tsart 라이브러리의 코어 기능을 담당하고 있고, tsart-{type}.js 파일은 사용하고자 하는 챠트가 구현되어 있다. 만일 막대 그래프를 사용하고 싶다면, tsart.js 파일을 먼저 선언하고 난 후, tsart-bar-standard.js 파일을 함께 선언하면 된다. 

<script src="...path/tsart.js"></script>
<script src="...path/tsart-bar-standard.js"></script>

차트가 그려질 콘테이너 역활을 담당할 HTML 요소를 생성하고,

<div id="chart-bar-01" style="width: 500px; height: 300px;"></div>

스크립트 요소 안에서 Tsart.draw 함수를 사용하여 챠트를 그리면 된다. 여기서 주의할 점은 그래프가 그려질 콘테이너는 반드시 가로, 세로 크기를 명시해야 한다는 것이다. 

<script>
/**
 * @param container 차트가 그려질 콘테이너의 DOM ID 또는 콘테이너 객체 
 * @param kind 차트 종류
 * @param style 차트 스타일
 * @param data 차트로 표현할 데이터 배열
 */
Tsart.draw("#chart-bar-01", "bar-standard", { /* style */ }, [
     { name: "철수", value: 100, group: "가", color: "#2eccfa" }
    ,{ name: "영희", value: 90, group: "가", color: "#2efffa" }
    ,{ name: "은혜", value: 94, group: "다", visible: false }
    ,{ name: "정석", value: 35, group: "다" }
    ...
]);
</script>

draw 함수의 첫 번째 파라미터는 차트가 그려질 콘테이너 요소의 DOM 아이디를 설정하거나 콘테이너 객체를 설정하면 된다., 두 번째 파라미터는 사용할 그래프의 종류를 설정하고, 세 번째 파라미터는 그래프 옵션을 설정하고, 마지막 파라미터는 그래프로 표현할 데이터를 설정한다.

두 번째 파라미터는 사용할 그래프의 종료를 나타내는 문자열을 설정한다.
